# coimbatore.ai - Premium Digital Experiences

Welcome to the **coimbatore.ai** project repository. We are a premier digital marketing and web development company based in Coimbatore, specialized in creating high-end, AI-powered digital experiences.

## Project Overview

This project is a modern React application built with:

- **Framework**: [Vite](https://vitejs.dev/) + [React](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI Architecture**: [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: Tailwind Animate + Custom CSS

## Getting Started

### Prerequisites

Ensure you have Node.js & npm installed.

### Installation

1. Clone the repository:
   ```bash
   git clone <YOUR_GIT_URL>
   cd luxe-digital-studio-main
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Development

- **Structure**:
  - `src/components`: Reusable UI components.
  - `src/pages`: Route pages (Home, About, Services, etc.).
  - `src/index.css`: Global styles and strict "Void" theme variables.

- **Theme**:
  - The project uses a strict "Void" black theme (`#000000` background).
  - Premium glassmorphism and animations are defined in `tailwind.config.ts`.

## Deployment

Build the project for production:

```bash
npm run build
```

This will generate a `dist` folder ready for deployment to Vercel, Netlify, or your preferred hosting provider.
